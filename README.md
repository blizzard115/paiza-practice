### 卵のペア問題

学んだこと

- `gets.split.map(&:to_i)` でスペース区切りの数字を受け取る
- `count` で配列の中にある数を数える
- 合計11になるペアは `1と10`, `2と9`, `3と8`, `4と7`, `5と6`
- `[count1, count2].min` で少ない方を取る

### 研究室の在室時間合計問題

学んだこと

- `"07:11".split(":")` で時刻を時間と分に分ける
- `hour * 60 + minute` で時刻を分に変換する
- 時刻の差分は「分」に直してから計算すると分かりやすい
- `/` で時間、`%` で余りの分を求める

### クラス委員の投票集計問題

学んだこと

- `Hash.new(0)` で初期値0のHashを作る
- `votes[name] += 1` で名前ごとの得票数を数える
- `max_by` で一番大きい値を持つ要素を探す
- Hashは「名前 → 票数」のような対応関係を管理するときに便利

### 母音を取り除くハンドルネーム生成問題

学んだこと

- `each_char` で文字列を1文字ずつ処理する
- `include?` で配列に指定した文字が含まれるか判定する
- `result += char` で文字を連結する
- `unless` は「条件に当てはまらない場合に実行する」処理
- `gsub` を使うと文字列の置換・削除もできる
