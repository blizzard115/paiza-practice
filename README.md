## Cランク

### 卵のペア問題

学んだこと

- `gets.split.map(&:to_i)` でスペース区切りの数字を受け取る
- `count` で配列の中にある数を数える
- 合計11になるペアは `1と10`, `2と9`, `3と8`, `4と7`, `5と6`
- `[count1, count2].min` で少ない方を取る

### 研究室の在室時間合計問題

学んだこと

- `"07:11".split(":")` で時刻を時間と分に分ける
- `hour * 60 + minute` で時刻を分に変換する
- 時刻の差分は「分」に直してから計算すると分かりやすい
- `/` で時間、`%` で余りの分を求める

### しりとり判定問題

#### 問題概要

N個の単語が順番に与えられる。  
前の単語の最後の文字と、次の単語の最初の文字がすべて一致していれば `Yes` を出力する。

途中で一致しない箇所があれば、初めて成立しなかった2文字を出力する。

- `gets.chomp` で文字列を受け取る
- 配列に単語を保存する
- `words[i]` と `words[i + 1]` で隣同士の単語を比較する
- `word[0]` で最初の文字を取得する
- `word[-1]` で最後の文字を取得する
- `exit` で条件に合わなかった時点で処理を終了する
